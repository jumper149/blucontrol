## Install

### Nix

Install blucontrol with the nix package manager:

```bash
git clone https://github.com/jumper149/blucontrol.git
cd blucontrol
nix-env -i -f default.nix
```

### Cabal

Build blucontrol with cabal-install:

```bash
git clone https://github.com/jumper149/blucontrol.git
cd blucontrol
cabal v2-build
```
