## Install

### Nix

Install bludigon with the nix package manager:

```bash
git clone https://github.com/jumper149/bludigon.git
cd bludigon
nix-env -i -f default.nix
```

### Cabal

Build bludigon with cabal-install:

```bash
git clone https://github.com/jumper149/bludigon.git
cd bludigon
cabal v2-build
```
