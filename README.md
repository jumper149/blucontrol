## Install

### Nix

Install blumon with the nix package manager:

```bash
git clone https://github.com/jumper149/blumon.git
cd blumon
nix-env -i -f default.nix
```

### Cabal

Build blumon with cabal-install:

```bash
git clone https://github.com/jumper149/blumon.git
cd blumon
cabal v2-build
```
